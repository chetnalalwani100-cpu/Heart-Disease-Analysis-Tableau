Interactive Tableau dashboard for analyzing heart disease risk factors and patient health trends.
# Heart Disease Analysis Dashboard

## Overview
This project analyzes heart disease data using Tableau to identify key health indicators and risk factors.

## Tools Used
- Tableau
- Excel

## Key Insights
- Heart disease trends across different age groups.
- Blood pressure and cholesterol analysis.
- Relationship between maximum heart rate and cholesterol.
- Distribution of health metrics among patients.
